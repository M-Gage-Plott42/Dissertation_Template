# Security Policy

## Reporting a vulnerability

Please report security issues privately via GitHub Security Advisories:

[Private security advisory submission](https://github.com/M-Gage-Plott42/Dissertation_Template/security/advisories/new)

## Repository hygiene

- Do not commit credentials, tokens, or local machine identifiers.
- Keep local build artifacts out of source control (see `.gitignore`).
- Treat the tracked UTC standards PDF/checksum/index as provenance artifacts;
  update them only when intentionally adopting a new official standards
  edition.
